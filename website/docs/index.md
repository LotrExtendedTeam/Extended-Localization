# 🌐 Localization Overview

This site provides a structured overview of the localization progress for the mod across all supported [Minecraft languages](https://minecraft.wiki/w/Language).  
Localization is tracked per file type, allowing contributors to easily identify areas that need attention.

## 🔄 Renewed Override Localization

This section documents ongoing efforts to improve and expand translations for content originally added by *LOTR Renewed*.

Many existing translations in Renewed are incomplete, inconsistent, or missing entirely.  
This project aims to deliver accurate, high-quality localization across all supported languages.

We also provide support for languages that were not originally included in Renewed.
